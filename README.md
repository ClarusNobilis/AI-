# Cyber Guard 安全管理系统

## 项目概述  
🛡️ 一款企业级网络安全管理系统，提供用户权限管理、安全工具集成和实时监控功能

## 主要功能  
✅ 多因子身份认证  
✅ 安全仪表盘  
✅ 漏洞扫描工具  
✅ 日志审计系统  
✅ 实时威胁监控

## 快速启动  
```bash
启动前端.bat
启动后端.bat
```

## 项目结构  
```
cyber-guard/
├── frontend/    # 前端界面
├── server/      # Node.js服务端
├── config/      # 系统配置
└── utils/       # 安全工具库
```

⚠️ 注意事项  
1. 首次使用需执行`npm install`安装依赖  
2. 数据库配置文件在`config/db.js`  
3. 认证密钥请参考`config/auth_codes.js`
